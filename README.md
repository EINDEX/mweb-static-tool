# MWeb static tool

![PyPI - Python Version](https://img.shields.io/pypi/pyversions/mweb-static-tool.svg)

将 Mweb 中的文章快速生成 Hexo 的文章。 用来方便使用更多模板的静态网站生成器。

**欢迎 PR 和 Issue**

## Usege

```bash
python3.6 main.py --site-name eindex.me --hexo-src <hexo-src> --mweb-src <mweb-src>
```

## 缺点

- Letex 需要主题支持
- 流程图时序图 需要主题支持
- mermaid 需要主题支持
